# CineAI - IA aplicada a la producción cinematográfica

Este repositorio contiene el archivo `cineai.html`, que podés publicar fácilmente con **GitHub Pages**.

## Cómo publicarlo

1. Creá un repositorio nuevo en GitHub (por ejemplo: `cineai`).
2. Subí el archivo `cineai.html` a la raíz del repositorio.
3. Opcional: subí también este `README.md`.
4. Andá a **Settings → Pages**.
5. En **Source**, elegí:
   - **Deploy from branch**
   - Branch: `main`
   - Folder: `/ (root)`
6. Guardá.

Tu página va a quedar accesible en una URL como:

`https://TU-USUARIO.github.io/NOMBRE-DEL-REPO/cineai.html`

Solo tenés que compartir ese enlace con tus alumnos.
